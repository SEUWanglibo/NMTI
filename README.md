# NMTI 牛马人格测试

基于 SBTI-test 改编的打工人版人格测试，专为职场牛马设计。

## 什么是 NMTI

NMTI (Niu Ma Type Indicator) 是一个娱乐性质的职场人格测试：

- **31道题目** 覆盖职场自我认知、情绪调节、价值取向、执行风格、人际关系五大维度
- **27种人格类型** 包括25种标准人格、1种兜底人格和1种隐藏人格
- **15维度评分** 详细解读你的牛马性格

## 人格类型一览

### 牛马生态位（职场身份类）
- **TRYOUT** - 试用期耗材：三个月转正，六个月毕业
- **FRESH** - 应届韭菜：刚被割，还在长
- **STALE** - 五年陈酿：2019年的经验，用到2024年
- **SANDWICH** - 夹心饼干：上面压，下面顶，中间碎
- **TARGET** - 高级背锅侠：工资高，锅更重，靶心更大
- **NOMAD** - 灵活就业难民：没社保，没年假，没明天
- **PLANT** - 工位盆栽：光合作用维生，存在感为零

### 牛马行为艺术（工作风格类）
- **EXCEL** - 表格雕花大师：把屎包装成巧克力，再雕个花
- **DONKEY** - 996永动驴：拉磨不求回报，停下就会思考
- **PIVOT** - 透视表农奴：被Excel奴役，为ROI献祭
- **PPT** - PPT缅北诈骗犯：只负责画饼，不管兑现
- **BRAIN** - 脑内亿万富翁：富可敌国的想象力，一毛钱的执行力
- **CV** - 简历缝补匠：东拼西凑，以次充好
- **SQUIRREL** - 知识松鼠症：囤积一切，消化为零
- **PLUGIN** - 生产力ED患者：工具买一堆，产出为0
- **ACTOR** - 会议室影帝：废话文学大师，黑话翻译机

### 牛马社交策略（人际关系类）
- **CUSHION** - 老板的情绪稳定器：人形沙包，随打随恢复
- **LONE** - 协作过敏体质：不会teamwork，只会solo送
- **CIA** - 茶水间情报局：情报贩子，八卦中间商
- **SHIELD** - 责任推卸奥运冠军：零失误，零担当，零存在感
- **VAMPIRE** - 成果嫁接专家：别人的树，你的果

### 牛马精神状态（心理状态类）
- **ZOMBIE** - 行尸走肉Plus：脑死亡，心还在跳
- **LOTTERY** - 年会抽奖妄想症：相信幸运会降临，年年落空年年信
- **LINKEDIN** - 脉脉自虐狂：越刷越焦虑，越焦虑越刷
- **EMPTY** - 电池健康度1%：充不进电，开不了机

### 特殊人格
- **AMOEBA** - 职场草履虫：单细胞生物，莫名存活（兜底人格）
- **NIGHT** - 深夜保安：公司是我家，我是看门狗（隐藏人格）
